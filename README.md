# xiang/homebrew-tap

Personal Homebrew Tap for custom macOS tools.

## 🛠 Available Formulae

### `makefs-darwin`
Native Darwin port of FreeBSD's `makefs` utility.

#### Installation
```bash
brew install xiang/tap/makefs-darwin
```

## 📦 Available Casks

### `wg-widget`
A simple macOS Menu Bar wrapper for `wg-quick`.

#### Installation
```bash
brew install --cask xiang/tap/wg-widget
```
